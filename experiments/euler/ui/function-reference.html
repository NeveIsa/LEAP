Core Euler Method Functions

euler_step(f, t, y, h)

Description: Perform one step of Euler's method. This is the core formula: y_{n+1} = y_n + h × f(t_n, y_n)

Parameters:

  • f (Callable): The derivative function f(t, y) = dy/dt
  • t (float): Current time
  • y (float): Current value
  • h (float): Step size

Returns: Tuple of (next_time, next_value)

Example:

# Perform one Euler step
next_t, next_y = euler_step(exponential_growth, 0.0, 1.0, 0.1)
# next_t = 0.1, next_y = 1.1

euler_method(f, y0, t0, h, n_steps)

Description: Complete Euler's method implementation (for reference). Students should try to implement their own version!

Parameters:

  • f (Callable): The derivative function f(t, y) = dy/dt
  • y0 (float): Initial value y(t0)
  • t0 (float): Initial time
  • h (float): Step size
  • n_steps (int): Number of steps to take

Returns: List of (t, y) pairs representing the solution

Derivative Functions

exponential_growth(t, y)

Description: Exponential growth: dy/dt = y

Parameters:

  • t (float): Time
  • y (float): Current value

Returns: float - The derivative dy/dt

exponential_decay(t, y)

Description: Exponential decay: dy/dt = -y

Parameters:

  • t (float): Time
  • y (float): Current value

Returns: float - The derivative dy/dt

logistic_growth(t, y, r=1.0, K=10.0)

Description: Logistic growth: dy/dt = r*y*(1 - y/K)

Parameters:

  • t (float): Time
  • y (float): Current value
  • r (float): Growth rate (default: 1.0)
  • K (float): Carrying capacity (default: 10.0)

Returns: float - The derivative dy/dt

harmonic_oscillator(t, y, omega=1.0)

Description: Simple harmonic oscillator: dy/dt = -ω² * y

Parameters:

  • t (float): Time
  • y (float): Current value
  • omega (float): Angular frequency (default: 1.0)

Returns: float - The derivative dy/dt

population_with_predation(t, y, r=0.5, a=0.1)

Description: Population with predation: dy/dt = r*y - a*y²

Parameters:

  • t (float): Time
  • y (float): Current value
  • r (float): Growth rate (default: 0.5)
  • a (float): Predation coefficient (default: 0.1)

Returns: float - The derivative dy/dt

Analysis Functions

compare_step_sizes(f, y0, t0, t_end, step_sizes)

Description: Compare Euler's method with different step sizes.

Parameters:

  • f (Callable): The derivative function f(t, y) = dy/dt
  • y0 (float): Initial value
  • t0 (float): Initial time
  • t_end (float): Final time
  • step_sizes (List[float]): List of step sizes to compare

Returns: Dictionary with step sizes as keys and solution lists as values

Example:

# Compare different step sizes
results = compare_step_sizes(exponential_growth, 1.0, 0.0, 2.0, [0.1, 0.05, 0.01])

calculate_error(true_solution, euler_solution)

Description: Calculate the error between true solution and Euler's method.

Parameters:

  • true_solution (Callable): Function that gives the true solution y(t)
  • euler_solution (List): List of (t, y) pairs from Euler's method

Returns: List of absolute errors at each time point

True Solution Functions

exponential_true_solution(t, y0=1.0)

Description: True solution for dy/dt = y: y(t) = y0 * e^t

Parameters:

  • t (float): Time
  • y0 (float): Initial value (default: 1.0)

Returns: float - The true solution value

exponential_decay_true_solution(t, y0=1.0)

Description: True solution for dy/dt = -y: y(t) = y0 * e^(-t)

Parameters:

  • t (float): Time
  • y0 (float): Initial value (default: 1.0)

Returns: float - The true solution value

Educational Functions

student_euler_solution(solution_data)

Description: Validate a student's Euler method implementation and provide feedback.

Parameters:

  • solution_data (dict): Dictionary containing:
    • derivative_function: The derivative function name
    • y0: Initial value
    • t0: Initial time
    • h: Step size
    • n_steps: Number of steps
    • student_solution: List of (t, y) pairs from student's implementation

Returns: Dictionary with validation results and feedback

get_euler_hint(step_number)

Description: Provide hints for implementing Euler's method step by step.

Parameters:

  • step_number (int): Which step the student is working on (0-based)

Returns: Hint string for that step

check_euler_formula(t, y, h, f_value, next_y)

Description: Check if a student's Euler formula implementation is correct.

Parameters:

  • t (float): Current time
  • y (float): Current value
  • h (float): Step size
  • f_value (float): Value of f(t, y)
  • next_y (float): Student's calculated next y value

Returns: Dictionary with feedback on correctness

Usage Examples

Basic Usage

# Solve exponential growth
solution = solve_exponential_growth(y0=1.0, h=0.1, n_steps=20)

# Solve logistic growth with custom parameters
solution = solve_logistic_growth(y0=2.0, h=0.05, n_steps=100, r=0.8, K=15.0)

# Compare step sizes
step_sizes = [0.1, 0.05, 0.01]
results = compare_step_sizes(exponential_growth, 1.0, 0.0, 2.0, step_sizes)

Error Analysis

# Solve with Euler's method
euler_solution = solve_exponential_growth(y0=1.0, h=0.1, n_steps=20)

# Calculate error against true solution
errors = calculate_error(exponential_true_solution, euler_solution)
